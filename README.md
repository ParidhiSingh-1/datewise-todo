# 📅 Datewise To‑Do List

A sleek, responsive To‑Do List web app built with **HTML, CSS, and JavaScript**.  
This project organizes tasks **date‑wise**, supports **dark/light themes**, and uses **local storage** for persistence. Designed with modern UI, smooth animations, and professional structure to deliver a polished productivity experience.

---

## 🚀 Features
- 📌 Add tasks with a chosen date
- 🗂️ Automatic date‑wise grouping
- 🌙 Dark/Light theme toggle (saved in local storage)
- 💾 Persistent storage (tasks remain after refresh)
- 🎨 Modern UI with smooth animations
- 📱 Responsive design for desktop and mobile
